# Perfect Prompt Sora Base

This local base is extracted from the repository `PerfectPromptSora.md` so the web app can reliably use it at runtime.

## Prompt Analysis Summary

### What Makes Sora 2 Prompts Go Viral?

1. **Novelty Factor**: Showcasing new Sora 2 features (audio, cameos, physics)
2. **Relatability**: Sports, ads, and pop culture references
3. **Meta-Content**: Self-referential content about AI itself
4. **Humor**: Parodies, absurdist juxtapositions, celebrity spoofs
5. **Technical Impressiveness**: Complex transformations, VFX, multi-layer compositions
6. **Practical Value**: Templates for real-world use cases (ads, content creation)

### Emerging Prompt Patterns

1. **Audio-First Prompts**: Explicitly requesting sounds, dialogue, music
2. **Style Shorthand**: Single phrases that trigger complex style recognition
3. **Sequential Specifications**: Breaking down transformations step-by-step
4. **Nested Realities**: Screens within screens, simulations within simulations
5. **Real-World References**: Celebrity names, brand names, current events

### Comparison: Sora 1 vs Sora 2 Viral Prompts

- Prompt length can be short or detailed.
- Explicit audio requests are now common.
- Multi-layer scenes and personalization are stronger in Sora 2.
- Physics and group dynamics are significantly improved.

## Tips for Creating Viral Sora 2 Prompts

1. **Leverage New Features**: Use audio, cameos, or enhanced physics
2. **Create Comparisons**: Show improvements over older tools
3. **Be Timely**: Reference current events or trends
4. **Add Humor**: Use unexpected combinations
5. **Show Practical Value**: Demonstrate business-ready use cases
6. **Go Meta**: Create content about creating content
7. **Target Communities**: Speak to specific audiences
8. **Use Recognizable Styles**: Leverage known visual languages

## Advanced Techniques

### Narrative with Cuts

- Use explicit transition tags like `[CUT TO]`.
- Define camera angle shifts between beats.
- Keep action sequence ordered and concise.
- Add location anchor details for coherence.

### Voiceover Recording Hack

- Place the scene in a voiceover-friendly environment when needed.
- Specify vocal style and delivery intention.
- Keep narration lines concise and rhythmic.
- Use this to improve perceived voice clarity.
